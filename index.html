Guide des ports

80
HTTP
Le port 80 est utiliser pour les communications HTTP (Hypertext Transfer Protocol). Il est generalement utiliser pour les connexions web et permet aux utilisateurs d'acceder aux sites web via un navigateur web.
443
HTTPS
Le port 443 est utiliser pour les communications HTTPS (Hypertext Transfer Protocol Secure). Il est generalement utilise pour les connexions web sécuriser et permet de securiser les donnees transmises entre le serveur web et le navigateur web.
22
SSH
Le port 22 est utiliser pour les connexions SSH (Secure Shell). Il permet une connexion securisee a un ordinateur distant et permet de gerer les fichiers et les processus sur cet ordinateur.
3306
MySQL
Le port 3306 est utilise pour les connexions MySQL. Il permet aux utilisateurs d'acceder a des bases de donnees MySQL via des applications ou des outils de gestion de base de donnees.

Exploit/payload

Toutes les infos sur les exploits, backdoors et payloads.

Payload

Un payload est un code malveillant qui est envoyé sur une cible pour causer des dommages ou voler des informations. Les attaquants peuvent utiliser des payloads pour prendre le contrôle d'un système, installer des logiciels malveillants ou voler des données.

Backdoor

Une backdoor est une méthode secrète pour accéder à un système ou un logiciel sans passer par les canaux normaux d'authentification ou de sécurité. Les backdoors sont souvent créées par des développeurs pour effectuer des tests, mais elles peuvent également être exploitées par des pirates pour accéder à des systèmes non autorisés.

Exploit

Un exploit est un code ou une technique qui exploite une vulnérabilité dans un système ou un logiciel pour causer des dommages ou prendre le contrôle de la machine. Les exploits peuvent être utilisés pour installer des logiciels malveillants, voler des informations ou accéder à un système non autorisé.

Attaque DDOS

Une attaque par déni de service distribué (DDOS) est une attaque où des pirates inondent un serveur ou un réseau avec des requêtes pour le rendre indisponible. Les attaquants utilisent souvent des botnets (réseaux d'ordinateurs infectés) pour mener des attaques DDOS massives.
